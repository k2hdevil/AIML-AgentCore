## 과정 목표

이 과정을 마치면 다음을 수행할 수 있게 됩니다:

1. **AWS 서비스를 사용하여 프로덕션에 바로 사용할 수 있는 에이전트를 구축합니다.**
2. **Amazon Bedrock AgentCore 서비스를 사용하여 에이전트를 배포합니다.**
3. **Amazon Bedrock AgentCore Gateway를 통해 에이전트 워크플로를 위한 여러 도구와 서비스를 통합합니다.**
4. **에이전트 인증 및 권한 부여를 위한 자격 증명 관리를 구성합니다.**
5. **에이전트 성능 최적화를 위한 모니터링 및 관찰성 패턴을 적용합니다.**

### 시간 조절 전략

**시간이 밀릴 때 (우선순위 순으로 축소)**:
1. 토론 주제 → 질문만 던지고 넘어감 (모듈당 5분 절약)
2. 지식 확인 문제 → 1문제만 풀고 나머지 자습 안내
3. 보충 내용 (Payments, Registry) → "이런 것도 있다" 한 줄 언급으로 축소
4. 실습 리뷰 → 15분으로 축소

**시간이 여유 있을 때**:
- 토론 주제 심화 (에이전트 설계 활동)
- 보충 내용 데모 (CLI로 Harness 실시간 생성)
- 추가 Q&A

## 실습 환경

### 요구 사항

| 항목 | 상세 |
|------|------|
| **운영 체제** | Windows, macOS, 또는 Linux (Ubuntu, SUSE, Red Hat) |
| **웹 브라우저** | Google Chrome, Mozilla Firefox, 또는 Microsoft Edge |
| **인터넷** | HTTPS 탐색 가능한 안정적 연결 |
| **실습 플랫폼** | AWS Builder Lab 등록 필요 |

### 실습 목표

실습에서는 다음을 시연합니다:
- AgentCore Memory를 사용하여 영구 메모리 기능 구현
- Lambda 함수가 포함된 AgentCore Gateway를 사용하여 중앙 집중식 도구 인프라 구축
- 보안 게이트웨이 액세스를 위한 JWT 기반 인증 구성
